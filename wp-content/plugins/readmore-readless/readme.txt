ReadMore ReadLess will hide content on page.  Clicking on readmore will show complete page without refreshing the page.

== Description ==

Show required content and let user click on read more.  Visitors will see readmore link on the page, when they will click, rest of the content appear with very simple animation on the same page.

**Features**

* This plugin will effect only on pages
* Select check box to show read more on the pages
* Control option shows on all pages, you can select to show readmore on the page
* Define the width to show from the page.
* Very simple and easy to use

== Installation ==

To do a new installation of the plugin, please follow these steps

1. Download the readmore-readless.zip file to your computer.
1. Unzip the file 
1. Upload 'readmore-readless' folder to the '/wp-content/plugins/'' directory
1. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= Does this works with all themes =

This plugin has been tested with Twenty Twelve, Twenty Thirteen and Twenty Fourteen theme and the plugin works like a charm with these themes.

= I have custom theme and the plugin does not show readmore ? =

You can wrap your content with "active" tags,  E.g. <active>Your content over here</active> and this will work.

= I have width of 300, the readmore link is not showing in the page ? =

To show readmore link on the page "Use readmore in this page" show be checked.
